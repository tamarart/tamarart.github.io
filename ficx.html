FicXixón

Redesigning the independent film festival.

Side Project. Barcelona 2017
Skills: UI, Art Direction.

About the project

FicXixón is a independet film festival in Spain. This project explores a better navigation and content architecture. Also, focuses on give more power to the graphic UI.

Focusing on users and content

Before starting anything, we spent time discussing the vision of the platform through a design thinking process. We wireframed and made low profile prototypes using marvel app, which we tested with users. We used sass, foundation and git to craft the code, and built a live style guide to help us to maintain a solid and reusable style in every element, following the approach of atomic design.
